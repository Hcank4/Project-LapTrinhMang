Mọi trang web bạn xem hàng ngày đều hoạt động dựa trên giao thức HTTP.
* **Request:** Trình duyệt gửi yêu cầu lấy dữ liệu.
* **Response:** Server gửi mã trạng thái (200 OK, 404 Not Found) và nội dung HTML.
